# Tic Tac Toe (Tkinter)

Integrantes: Adriano

Juego de Tic Tac Toe (3 en raya) desarrollado en Python con interfaz gráfica usando Tkinter.

## Funcionalidades
- Turnos X y O
- Detección de ganador (filas, columnas y diagonales)
- Detección de empate
- Botón para reiniciar el juego sin cerrar la aplicación
- Interfaz visual mejorada (colores y tipografía)

## Requisitos
- Python 3.11 o superior
- Tkinter (incluido con Python)
